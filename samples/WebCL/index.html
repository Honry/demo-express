This sample demonstrates the basic WebCL functionality.

Requirement:
  • This sample requires OpenCL library support.
  • On IA side, PowerVR OpenCL library is installed on CloverTrail devices by default(e.g. ZTE geek, K900 etc).
  • Intel OpenCL library is installed from the Baytrail devices by default (e.g. Asus MeMopad, Ramo i9/10 etc).
  • On ARM side, this library is provided by Qualcomm from the Snapdragon 600 chipset by default (e.g. Galaxy S4, Xiaomi3 etc).

This sample is designed by WebCL, WebCLPlatform, WebCLDevice, WebCLContext and WebCLCommandQueue interface.
